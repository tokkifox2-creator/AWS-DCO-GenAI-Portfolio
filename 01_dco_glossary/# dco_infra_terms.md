## 2. 내가 만든 DCO 인프라 용어 정의

### Rack

내 정의:수많은 컴퓨터(서버)와 네트워크 장비들을 차곡차곡 쌓아 올릴 수 있도록 만든 커다란 철제 캐비닛입니다. 도서관의 책장이나 아파트 건물과 같은 역할을 합니다.

### Server

내 정의:우리가 흔히 아는 개인용 PC에서 모니터와 키보드를 없애고, 오직 데이터 저장과 처리 능력만 극대화하여 24시간 쉬지 않고 일하도록 만든 특수 컴퓨터입니다.

### NIC

내 정의:퓨터가 인터넷이나 다른 컴퓨터와 통신할 수 있게 랜(LAN)선을 꽂는 부품입니다. 흔히 '랜카드'라고 부르는 부품의 정식 명칭입니다.

### ToR Switch

내 정의:하나의 랙(아파트) 안에 살고 있는 여러 대의 서버(주민)들이 외부 인터넷망으로 나갈 수 있도록 선들을 한데 모아주는 '네트워크 분배기'입니다. 주로 랙의 맨 꼭대기(Top)에 위치해서 이런 이름이 붙었습니다.

### Link Down

내 정의:장비와 장비 사이를 잇는 통신선이 완전히 끊어져 서로 신호를 주고받지 못하는 상태를 말합니다. 전원 코드가 뽑힌 것과 비슷한 상황입니다.

### CRC error

내 정의:컴퓨터끼리 데이터를 주고받는 통신 과정에서 데이터가 깨지거나 훼손되어 도착했음을 알려주는 오류입니다. 택배를 받았는데 상자가 심하게 구겨져 있거나 내용물이 손상된 것과 같습니다.

### SSD failure

내 정의:데이터를 영구적으로 저장하는 플래시 메모리 기반의 저장 장치(SSD)가 수명을 다하거나 고장 나서 더 이상 데이터를 읽고 쓸 수 없는 상태입니다.

### Fan Alert

내 정의:뜨거운 서버 내부를 식혀주는 선풍기(쿨링 팬)가 멈췄거나 비정상적으로 느리게 돌아갈 때 울리는 경고 신호입니다.
